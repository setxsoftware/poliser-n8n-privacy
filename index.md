# Política de privacidad – Automatizaciones internas (Poliser)

Esta aplicación es utilizada exclusivamente para automatizaciones internas de la empresa Poliser.

La aplicación accede a servicios de Microsoft 365 (Outlook, Excel, OneDrive) únicamente para la ejecución de procesos operativos internos.

Los datos procesados no son compartidos con terceros ni utilizados con fines comerciales externos.

La información se mantiene dentro del entorno de Microsoft 365 de la empresa y del sistema de automatización n8n.
